GW2API
======
A high level access library for the Guild Wars 2 API, aiming for organized output.

#### Features:
* Items
  * Basic
    * Bag
    * Consumable
    * CraftingMaterial
    * Gathering
    * Gizmo
    * MiniPet
    * Tool
    * UpgradeComponent
  * Gear
    * Armor
    * Back
    * Trinket
    * Weapon
